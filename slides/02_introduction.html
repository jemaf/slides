Hey there!

This repository is an extension of reveal.js project enriched with new features and styles that I use to give my classes/courses.

Feel free to use/change it in the way that you prefer!

Style

A new style was created specific for academic purposes!

The idea is to use it both in classes and long-term courses.

Typography

I selected a more academic typography:

  • Fredericka the Great is used for titles and headings
  • Josefin Slab is used for slides body
